# Child Care Foundation Nigeria - NGO Website

![ICFC Landing Page Screenshot](https://icfc-ngo.vercel.app/images/screenshot.webp)

A responsive landing page for **Child Care Foundation**, an NGO in Nigeria dedicated to improving the lives of vulnerable children.

**Live Demo:** (https://icfc-ngo.vercel.app/)

---

## Problem

Many NGOs in Nigeria struggle to reach a wider audience online. Donors, volunteers, and the public often find it difficult to access reliable information about projects, donation opportunities, and the impact of their contributions.

---

## Solution

This website provides a **centralized, engaging, and informative platform** that:

- Clearly communicates the NGOâ€™s mission and impact
- Showcases ongoing and past projects
- Provides an easy way for visitors to get involved or donate
- Creates trust through professional design and transparency

---

## Features

- Fully responsive design for desktop, tablet, and mobile
- Modern UI using **Chakra UI** and **Tailwind CSS**
- Built with **React** and **Next.js** for performance and SEO
- Sections: About Us, Projects, Get Involved, Contact
- Smooth animations and interactive elements for better UX

---

## Tech Stack

- **Frontend:** React, Next.js
- **Styling:** Chakra UI, Tailwind CSS
- **Deployment:** Vercel
